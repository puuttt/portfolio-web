# Portfolio Next.js

This is a portfolio website built with Next.js and Tailwind CSS, inspired by Daniel Herzog's design.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/page.tsx`: The main landing page.
- `src/components`: Contains all the UI components (Navbar, Hero, About, Services, Projects, Process, Testimonials, Footer).
- `src/app/globals.css`: Global styles and Tailwind directives.

## Features

- Responsive design
- Dark/Light mode support (system preference)
- Smooth scrolling
- Clean and minimalist aesthetic
